# Конфигурация NixOS (v24.11)

Этот конфиг создан только для личного использования, но может служить примером для создания своего собственного конфига.

## Система

- **Устройство:** Ноутбук с графикой Intel и NVIDIA GT 650M.
- **Драйверы:** NVIDIA 470. Так как Wayland нестабилен с NVIDIA, использую Xorg с оконным менеджером **Qtile**.
- **DPI:** Для X-сервера установлен DPI 92.
- **Менеджер входа:** SDDM с темой *Astronaut*.
- **Bluetooth:** Хотя ноутбук поддерживает Bluetooth, для моей модели NixOS подбирает неверные драйверы, поэтому используется опция `enableAllFirmware`.

## Файловая система

1. **FAT32** (0.5 GB) для GRUB.
2. **ext4** (109 GB) для корневого раздела. В `/home` примонтирован диск с проектами и играми.
3. **Swap:** 9 GB.

## Программное обеспечение и инструменты

- **Языки программирования:** Python, Rust и Node.js (с дополнительными пакетами).
- **Утилиты:** Основной – `doas`, также имеется `sudo`.
- **Очистка системы:** Автоматическое удаление пакетов и поколений NixOS (ежедневно удаляются поколения старше двух дней).
- **Микрокод:** Установлен пакет `ucode-intel`.
- **Локализация:** Турецкое время и английский язык.
- **Сеть:** Использую NetworkManager.
- **Игры:** Иногда играю в Minecraft с Prism Launcher.
- **Аудио:** Pipewire.
- **Удалённый доступ:** SSH.
- **Редакторы:** VSCode для разработки и nano для быстрых правок.
- **Терминал и оболочка:** Alacritty (из коробки выглядит привлекательно) и zsh (иногда с tmux).
- **Дополнительно:** Transmission для торрентов, Lutris для игр, браузеры Chrome и Firefox.
